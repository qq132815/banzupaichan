# 产品定义模块 - 导入导出功能说明

## 功能概述

已成功为产品定义模块添加了完整的导入导出功能，包括：

### 1. 导出功能
- **接口**: `GET /api/export/product_definitions`
- **功能**: 将产品定义数据导出为Excel文件
- **字段**: 产品编码、产品名称、规格型号、分类、客户、项目、描述、状态、图片URL

### 2. 导入功能
- **接口**: `POST /api/import/product_definitions`
- **功能**: 从Excel文件导入产品定义数据
- **支持**: 自动下载图片（如果提供了图片URL）
- **特性**: 
  - 自动匹配表头（支持多种表头名称）
  - 使用 INSERT OR REPLACE 实现更新或插入
  - 自动下载并保存产品图片

### 3. 模板下载
- **功能**: 下载CSV格式的导入模板
- **字段**: 产品编码、产品名称、规格型号、分类、客户、项目、描述、状态、图片URL

## 使用说明

### 导出数据
1. 访问产品定义页面
2. 点击"📥 导出"按钮
3. 自动下载Excel文件

### 导入数据
1. 点击"📄 模板"下载导入模板
2. 按照模板格式填写数据
3. 点击"📤 导入"按钮
4. 选择Excel文件上传
5. 系统自动处理并导入数据

### 注意事项
- 导入时会自动下载图片（如果提供了URL）
- 图片保存在 `static/images/products/` 目录
- 使用 INSERT OR REPLACE 策略，相同产品编码会更新
- 支持多种表头名称匹配
