# 排班界面产品图片显示功能 - 使用说明

## ✅ 功能已实现

### 功能概述
在排班界面的新增排班弹窗中，右侧添加了一个产品图片显示区域。当选择工单后，系统会自动根据产品编码匹配并显示产品图片。

### 如何使用

#### 1. 访问排班页面
```
http://localhost:5000/schedule-page
```

#### 2. 点击"＋ 新增排班"按钮
- 页面顶部工具栏有"＋ 新增排班"按钮
- 点击后弹出新增排班弹窗

#### 3. 查看产品图片区域
- 弹窗现在分为左右两栏
- 左侧：原有的表单区域
- 右侧：产品图片显示区域（250px宽）

#### 4. 选择工单
- 在搜索框中输入产品名称、工单号或产品编码
- 从搜索结果中选择一个工单
- 右侧会自动显示该产品的图片

### 功能特性

#### 图片显示区域
- 位置：弹窗右侧
- 大小：250px宽，自适应高度
- 样式：圆角边框，灰色背景
- 默认显示："选择工单后显示产品图片"

#### 触发场景
1. **点击"＋ 新增排班"**：点击按钮后弹窗显示，选择工单后显示图片
2. **双击甘特图空白区域**：双击后弹窗显示，选择工单后显示图片
3. **预警一键排班**：在预警面板点击"一键排班"，自动填充工单并显示图片

#### 图片预览
- 点击产品图片，弹出全屏预览
- 点击预览图片外部区域或"关闭"按钮关闭

### 显示逻辑

1. **默认状态**：显示占位符"选择工单后显示产品图片"
2. **选择工单后**：
   - 如果产品有图片（image_path或image_url），显示图片
   - 如果产品没有图片，显示"未找到产品图片"
3. **重置表单**：清空图片显示，恢复默认占位符

### 技术实现

#### 前端修改
1. **弹窗布局**：宽度从600px增加到900px，分为左右两栏
2. **新增函数**：
   - `loadProductImage(productCode)` - 加载产品图片
   - `showImagePreview(src)` - 显示图片预览
3. **修改函数**：
   - `selectOrder(o)` - 选择工单后加载图片
   - `resetForm()` - 重置表单时清空图片

#### 后端API
使用现有的产品定义API：
```
GET /api/product-definitions?q={product_code}&page_size=1
```

### 故障排除

#### 问题1：看不到图片显示区域
**解决方法**：
1. 清除浏览器缓存（Ctrl+F5）
2. 重新访问排班页面
3. 点击"＋ 新增排班"查看弹窗

#### 问题2：图片不显示
**可能原因**：
- 产品没有图片
- 图片URL失效
- 网络连接问题

**解决方法**：
1. 检查产品定义中是否有图片
2. 检查图片URL是否有效
3. 检查网络连接

#### 问题3：弹窗布局异常
**解决方法**：
1. 清除浏览器缓存
2. 刷新页面
3. 检查浏览器控制台是否有错误
