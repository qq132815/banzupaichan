# 产品定义模块 - 完整功能说明

## 功能概述

已成功实现产品定义模块，支持从实际Excel文件导入产品数据。

## 实际数据导入

### Excel文件结构
- **文件**: `产品信息_全部.xlsx`
- **总行数**: 5349行（含表头）
- **实际数据**: 5347条产品记录

### 表头字段映射
实际表头 → 数据库字段：

1. **产品类型** → `product_type`
2. **产品编号** → `product_code`（必填）
3. **产品名称** → `product_name`
4. **产品规格** → `specifications`
5. **单位** → `unit`
6. **工艺路线** → `route_code`
7. **最小安全库存** → `safety_stock`
8. **最大安全库存** → （暂未映射）
9. **库存数量** → `stock_qty`
10. **产品来源** → `source`
11. **产品图** → `image_url`（自动下载）
12. **客户** → `customer`
13. **每筐容量** → `basket_capacity`

## 数据库字段

products表新增字段：
- `product_type` TEXT - 产品类型（成品/半成品/物料）
- `unit` TEXT - 单位
- `route_code` TEXT - 工艺路线代码
- `stock_qty` REAL - 库存数量
- `source` TEXT - 产品来源
- `basket_capacity` REAL - 每筐容量

## 功能特性

### 1. 智能表头匹配
- 自动识别第二行为表头（第一行是合并单元格标题）
- 支持多种表头名称变体

### 2. 图片自动下载
- 导入时自动下载产品图片
- 图片保存在 `static/images/products/` 目录
- 文件名使用产品编码（特殊字符替换）

### 3. 数据更新策略
- 使用 INSERT OR REPLACE 策略
- 相同产品编码会更新数据

## 使用说明

### 导入产品数据
1. 访问产品定义页面
2. 点击"📤 导入"按钮
3. 选择 `产品信息_全部.xlsx` 文件
4. 系统自动处理并导入数据
5. 图片会自动下载并保存

### 导出产品数据
1. 点击"📥 导出"按钮
2. 下载Excel文件

### 下载模板
1. 点击"📄 模板"按钮
2. 下载CSV格式模板

## 文件结构

- `app.py` - API路由（导入/导出/CRUD）
- `utils/excel.py` - Excel导入函数
- `templates/product_definitions.html` - 前端页面
- `产品信息_全部.xlsx` - 实际数据文件

## 注意事项

1. **图片下载**: 导入时会自动下载图片，需要网络连接
2. **文件格式**: Excel文件第一行是标题，第二行是表头
3. **数据量**: 支持5000+条数据导入
4. **字符编码**: 支持中文字符
