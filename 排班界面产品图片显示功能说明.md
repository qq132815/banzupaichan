# 排班界面产品图片显示功能

## 功能概述

在排班界面的新增排班弹窗中，右侧添加了一个产品图片显示区域。当选择工单后，系统会自动根据产品编码匹配并显示产品图片。

## 功能特性

### 1. 图片显示区域
- 位置：新增排班弹窗的右侧
- 大小：250px宽，自适应高度
- 样式：圆角边框，灰色背景

### 2. 触发场景
- **双击新增**：双击甘特图空白区域新增排班时
- **点击新增**：点击"＋ 新增排班"按钮时
- **预警一键排班**：在预警面板点击"一键排班"按钮时

### 3. 显示逻辑
1. 默认显示占位符："选择工单后显示产品图片"
2. 选择工单后，自动查询产品图片
3. 如果产品有图片（image_path或image_url），显示图片
4. 如果产品没有图片，显示"未找到产品图片"
5. 点击图片可以放大预览

## 技术实现

### 前端修改

#### 1. 弹窗布局修改
- 将弹窗宽度从默认600px增加到900px
- 弹窗内容分为左右两栏：
  - 左侧：原有表单区域（flex:1）
  - 右侧：图片显示区域（250px固定宽度）

```javascript
// 加载产品图片

// 显示图片预览（页面内模态框）

```

#### 3. 修改的函数
- `selectOrder(o)`：选择工单后调用 `loadProductImage(o.product_code)`
- `resetForm()`：重置表单时调用 `loadProductImage(null)` 清空图片

### 后端API

使用现有的产品定义API：
```
GET /api/product-definitions?q={product_code}&page_size=1
```

返回数据包含：
- `image_path`：本地图片路径
- `image_url`：原始图片URL

## 使用说明

### 场景1：点击"＋ 新增排班"
1. 点击页面上的"＋ 新增排班"按钮
2. 弹窗右侧显示"选择工单后显示产品图片"
3. 搜索并选择工单
4. 右侧自动显示该产品的图片

### 场景2：双击甘特图空白区域
1. 在甘特图上双击空白区域
2. 弹窗自动填充时间和设备信息
3. 搜索并选择工单
4. 右侧自动显示该产品的图片

### 场景3：预警一键排班
1. 切换到"⚠️ 预警"标签
2. 找到需要排班的预警项
3. 点击"一键排班"按钮
4. 弹窗自动填充工单信息
5. 右侧自动显示该产品的图片

### 图片预览
- 点击产品图片，弹出全屏预览
- 点击预览图片外部区域或"关闭"按钮关闭

## 注意事项

1. **图片来源**：优先使用 `image_path`（本地图片），其次使用 `image_url`（远程图片）
2. **性能考虑**：每次选择工单都会查询产品图片，但查询很快（单条记录）
3. **兼容性**：如果没有图片，显示友好的占位符提示
4. **预览功能**：点击图片可以放大预览，使用页面内模态框，不打开新窗口
