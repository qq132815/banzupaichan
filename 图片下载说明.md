# 产品图片下载说明

## 当前状态

✅ **数据导入完成**
- 总产品数: 5,347条
- 有图片URL: 1,667个
- 已下载图片: 0个

## 图片下载选项

### 选项1: 导入时自动下载（已启用）
当前导入功能会在导入数据时自动下载图片。

### 选项2: 批量下载脚本
已创建批量下载脚本: `scripts/download_all_images.py`

**使用方法:**
```bash
python scripts/download_all_images.py
```

**功能:**
- 自动下载所有产品图片
- 跳过已下载的图片
- 显示下载进度
- 更新数据库中的image_path字段

### 选项3: 手动下载
可以通过产品定义页面的"下载图片"按钮逐个下载。

## 图片存储位置

下载的图片保存在: `static/images/products/`

文件命名规则:
- 使用产品编码作为文件名
- 特殊字符替换为下划线
- 保留原始文件扩展名

## 注意事项

1. **网络要求**: 下载图片需要网络连接
2. **时间消耗**: 1667个图片预计需要10-20分钟
3. **磁盘空间**: 预计需要100-500MB空间
4. **错误处理**: 失败的图片会记录日志

## 建议

如果图片不是立即需要，可以：
1. 先使用系统功能
2. 空闲时运行批量下载脚本
3. 或者在产品详情页逐个下载需要的图片
