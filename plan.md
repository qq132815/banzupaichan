# plan.md - 当前开发进度

## 最近更新 (2026-05-29)

### 已完成功能
1. ✅ 用户认证系统
   - 登录/登出
   - 角色权限(planner/team)
   - 班组隔离

2. ✅ 甘特图排班
   - 设备行显示(绿色标识有排产)
   - 任务条显示产品名称+工序+数量+人员
   - 左右拖拽调整时间(30分钟吸附)
   - 上下拖拽更换设备
   - 单击编辑、双击新增
   - 悬停信息跟随鼠标

3. ✅ 工单工序进度
   - 解析工单的process_progress字段
   - 进度卡片显示(颜色区分完成度)
   - 点击卡片选择工序
   - 工序下拉框只显示工单工序
   - 自动填充剩余数量

4. ✅ 产品管理
   - 工艺路线(支持导入/导出/编辑)
   - 生产周期(支持导入/导出/编辑)
   - 物料清单(支持导入/导出/编辑)

5. ✅ 计划审批流程
   - 草稿→提交→审批
   - 退回修改功能

### 待优化

- [ ] 更多预警规则
- [ ] 批量排班功能
