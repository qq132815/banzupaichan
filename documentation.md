## 二、计划员功能

### 2.1 导入发货计划
1. 进入"数据导入"页面
2. 选择"发货计划"类型
3. 上传Excel文件
4. 系统自动导入并倒推计算

### 2.2 审批计划
1. 进入"计划审批"页面
2. 查看各班组提交的计划
3. 点击"通过"或"退回"
4. 退回需填写原因

### 2.3 查看预警
1. 进入"预警中心"页面
2. 红色: 紧急(3天内)
3. 黄色: 关注(7天内)
4. 绿色: 正常

## 三、班组功能

### 3.1 排班操作
1. 选择班组和日期
2. 点击"新增排班"
3. 搜索工单号或产品编码
4. 选择工单后自动显示工序进度
5. 选择工序和设备
6. 填写生产人员
7. 调整时间(可拖拽甘特图)
8. 保存

### 3.2 工序进度说明
选择工单后，系统显示该工单的工序进度:
- 绿色卡片: 完成度>=80%
- 黄色卡片: 完成度>=50%
- 红色卡片: 完成度<50%
- 点击卡片可快速选择工序

### 3.3 甘特图操作
- 拖拽任务条左右边缘: 调整时间
- 拖拽任务条中间: 移动到其他设备
- 单击任务条: 打开编辑窗口
- 双击空白区域: 新增排班
- 悬停查看详细信息

### 3.4 提交计划
1. 排班完成后点击"提交计划"
2. 计划状态变为"待审批"
3. 等待计划员审批
4. 如被退回，修改后重新提交

## 四、产品管理

### 4.1 工艺路线
- 查看: 进入"产品管理-工艺路线"
- 导入: 点击"导入"上传Excel
- 导出: 点击"导出"下载Excel
- 编辑: 点击行内"编辑"按钮

### 4.2 生产周期
- 管理各产品的生产周期和提前期
- 用于发货计划倒推计算

### 4.3 物料清单
- 管理成品与半成品的物料关系
- 支持导入/导出/编辑

## 六、常见问题

### Q: 为什么工序下拉框没有数据?
A: 请确认:
1. 该产品已有工艺路线
2. 工艺路线中的工序名称与工单进度中的名称一致

### Q: 为什么不能加班?
A: 加班功能仅限当日使用，非当日无法勾选加班。

### Q: 计划被退回后如何修改?
A: 在排班页面可以看到"已退回"状态，直接修改后重新提交。

### Q: 如何重建数据库?
A: 运行命令: python scripts/rebuild_all.py
