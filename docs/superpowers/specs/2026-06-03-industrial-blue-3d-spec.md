# 3D 可视化工作台 - 资源升级方案

## 1. 现状分析
目前的 MVP 使用了简单的 Three.js 代码，虽然加了 Bloom 但还比较基础。要做到“成熟”且符合“现代工业蓝”风格，最高效的方法不是从零造轮子，而是 **借鉴成熟的开源 Demo 架构** 并 **引入高精度模型资产**。

## 2. “工业蓝”视觉特征
- **材质**: 
  - **金属拉丝**: 设备主体使用 PBR 材质，带有各向异性高光。
  - **磨砂玻璃**: 设备的操作面板或防护罩使用半透明材质。
  - **发光描边**: 关键部件带有蓝色能量描边 (Edge Glow)。
- **光照**: 
  - **环境光遮蔽 (AO)**: 增加阴影细节，让设备更有立体感。
  - **体积光 (Volumetric)**: 模拟丁达尔效应，增加空间层次。

## 3. 开源资源推荐
### A. GitHub 仓库 (Demo 借鉴)
- **Three.js Smart Factory**: 很多大佬做的智慧工厂 Demo，通常包含传送带、机械臂、数据面板。
- **Three.js Digital Twin**: 数字孪生项目，通常侧重于数据驱动动画。

### B. 3D 模型资源 (资产)
- **Poly Pizza / Sketchfab**: 提供大量 CC0 (免费商用) 的工业模型。
- **关键模型**: 
  - CNC 数控机床 (CNC Machine)
  - 工业机械臂 (Robotic Arm)
  - 自动导引车 (AGV)
  - 传送带 (Conveyor Belt)

## 4. 实施建议
1.  **风格重构**: 重写 Shader 或材质系统，统一为“深蓝 + 科技白 + 能量青”配色。
2.  **模型替换**: 不再使用 Box 几何体，而是加载真实的 GLB 模型。
3.  **交互升级**: 增加 **OutlinePass** (描边) 和 **Lensflare** (镜头光晕)，提升质感。
