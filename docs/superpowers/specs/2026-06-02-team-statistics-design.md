# 班组统计模块设计文档

日期: 2026-06-02
状态: 待实现

## 1. 概述

将现有"完成率统计"页面升级为"班组统计"模块，实现4个班组之间的多维度数据对比，支持日/周/月三种时间粒度，通过卡片摘要、图表可视化、明细表格三种形式呈现。

## 2. 变更范围

- `templates/statistics.html` — 完全重写
- `app.py` — 新增3个API路由，替换原有 `/api/statistics`
- `base.html` — 侧边栏文字"完成率统计"改为"班组统计"

## 4. API设计

### 4.1 GET /api/stats/daily

参数:
- `date_from` (必填): 起始日期 YYYY-MM-DD
- `date_to` (必填): 结束日期 YYYY-MM-DD

返回:

  "date_to": "2026-06-01",
  "teams": [

      "id": 1,
      "name": "前段",
      "member_count": 33,
      "summary": {
        "completion_rate": 92.5,
        "production_efficiency": 88.3,
        "good_rate": 99.8,
        "overtime_ratio": 25.0,
        "utilization_rate": 76.5,

      "days": [

### 4.2 GET /api/stats/weekly

参数:
- `week_start` (必填): 周一日期 YYYY-MM-DD

返回: 结构同daily，days替换为weeks数组，每个元素包含week_start/week_end及汇总指标。

### 4.3 GET /api/stats/monthly

参数:
- `month` (必填): 月份 YYYY-MM

返回: 结构同daily，days替换为months数组，每个元素包含month及汇总指标。

## 5. 指标计算公式

| 指标 | 公式 | 数据来源 |
|------|------|----------|
| 完成率 | completed_qty / planned_qty x 100 | schedules + work_reports |
| 生产效率 | AVG(work_reports.efficiency) WHERE efficiency > 0 | work_reports.efficiency |
| 良品率 | SUM(good_qty) / SUM(report_qty) x 100 | work_reports |
| 加班人数比例 | COUNT(DISTINCT overtime users) / COUNT(DISTINCT all users) x 100 | attendance.is_overtime |
| 工时利用率 | SUM(report_hours) / SUM(work_hours) x 100 | work_reports + attendance |
| 人均产量 | total_qty / 出勤人数 | work_reports + attendance |
| 计划量 | SUM(schedules.quantity) | schedules |
| 完成量 | SUM(work_reports.report_qty) | work_reports |
| 出勤天数 | COUNT(DISTINCT attendance records) | attendance |

### 班组-工序映射

通过 processes 表的 team_name 字段关联:
- team_name 包含班组名称的工序归属该班组
- work_reports 通过 process_name 关联到 processes，再映射到 team_id

### KPI卡片汇总值

取日期范围内加权平均:
- 完成率 = 总完成量 / 总计划量 x 100
- 其他指标 = 各天汇总后的加权平均

## 6. 智能选择器

根据当前Tab动态切换日期控件:
- 日视图: 两个date input (date_from, date_to)，默认近7天
- 周视图: 一个date input (选择周一)，显示周范围提示
- 月视图: 一个month input (选择月份)

## 7. 图表设计

### 柱状图（班组对比）

- X轴: 4个班组名称
- Y轴: 指标值（百分比）
- 支持切换指标: 完成率 | 生产效率 | 良品率 | 加班比例 | 工时利用率
- 默认显示完成率
- 颜色: 4个班组各用不同颜色

### 折线图（时间趋势）

- X轴: 日期 / 周 / 月
- 多条线: 每个班组一条，颜色与柱状图一致
- 默认显示完成率趋势
- 图例点击可隐藏/显示某个班组

### 图表切换控件

柱状图和折线图共用一组按钮:
```
[完成率] [生产效率] [良品率] [加班比例] [工时利用率]
```
点击同时更新两个图表。

## 8. KPI卡片设计

5张卡片，每张包含:
- 图标
- 指标名称
- 数值（百分比）
- 趋势箭头（较上一周期变化）

颜色编码:
- 绿色(>=目标值)
- 橙色(80-99%)
- 红色(<80%)

目标值从 system_settings 读取，无配置时用默认阈值。

## 9. 明细表格设计

### 表格布局

行 = 4个班组，列 = 指标参数

日视图:
| 班组 | 完成率 | 生产效率 | 良品率 | 加班比例 | 工时利用 | 计划量 | 完成量 |
|------|--------|----------|--------|----------|----------|--------|--------|

周视图:
| 班组 | 完成率 | 生产效率 | 良品率 | 加班比例 | 工时利用 | 计划量 | 完成量 | 出勤天数 |
|------|--------|----------|--------|----------|----------|--------|--------|----------|

月视图:
| 班组 | 完成率 | 生产效率 | 良品率 | 加班比例 | 工时利用 | 计划量 | 完成量 | 出勤天数 | 人均产量 |
|------|--------|----------|--------|----------|----------|--------|--------|----------|----------|

### 表格特性

- 数值颜色编码（绿/橙/红）
- 点击某行 → 高亮该班组，图表联动聚焦
- 表格底部加汇总行（全公司均值）
- 无数据单元格显示灰色 "—"
